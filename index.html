What is remoteStorage?

Simple answer: It's an open storage specification enabling people to have one common data storage for use with their apps.

Technical answer: It's Discovery (e.g. Webfinger) + Read-Write-Web Personal Data Service + CORS. It works great with unhosted web apps. Find more info on remotestorage.io

What is remoteStorage.js?

It's a JavaScript library, which makes your app talk to remoteStorage.

Browser support

This library relies heavily on CORS (Cross-Origin Resource Sharing).

Known to work: Firefox, Chrome, Safari, Safari iOS.

Might work: Firefox Mobile, Android stock browser, Opera 12+, Opera Mobile 12+, IE 10+.
